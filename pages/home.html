Meus Projetos

14° Desafio da Computação - Python Classroom

O Python Classroom é um aplicativo mobile desenvolvido em Flutter, um framework de código aberto da Google. O projeto teve como objetivo ensinar os conceitos básicos da linguagem de programação Python, oferecendo exercícios práticos e referências de estudo.

15° Desafio da Computação - Fechadura Eletrônica Inteligente

A Fechadura Eletrônica Inteligente é um projeto que integrou hardware e software de maneira eficiente, permitindo o controle de uma trava que é destravada por sinais RFID de tags configuradas ou dispositivos móveis com NFC cadastrados.

16° Desafio da Computação - Battery Tracker

O Battery Tracker é um aplicativo mobile desenvolvido em React Native, um framework de código aberto da Meta. O objetivo do projeto foi ensinar sobre o funcionamento de baterias em dispositivos modernos, explicando como elas operam e como tratá-las corretamente. Além disso, o aplicativo fornece insights e informações sobre a bateria do smartphone.

17° Desafio da Computação - Dome

O Dome é uma aplicação mobile desenvolvida em React Native, que, em conjunto com uma ESP-32 programada em MicroPython, permite o controle interativo de ventiladores e lâmpadas de forma inteligente. O aplicativo é capaz de controlar esses dispositivos de maneira eficiente.
